# E-Commerce Platform - Next.js Storefront

### Used technologies

- [Next.js 14 (using app directory)](https://nextjs.org/)
- [React](https://reactjs.org/)
- [Mantine](https://mantine.dev/)

## Installation

```bash
npm install
```

## Running the app in development mode

```bash
npm run dev
```

## Building the app

```bash
npm run build
```

## Starting the app

```bash
npm run start
```
